# cherry

## 简介
Cherry是一个简洁、方便且易用的Java后端开发工具框架，涵盖了IO、中间件、多线程、微服务、Excel等内容。

该框架主要由个人开发经验积累编写、仅为日常开发提供代码Demon和开发思路。

## 包含组件

| 模块         | 介绍                                   |
| ------------ | -------------------------------------- |
| cherry-core  | 框架核心，包含了通用工具类、实体模型等 |
| cherry-excel | Excel处理、易用工具类                  |
